# UX DataTables

UX DataTables is a Symfony bundle integrating the [DataTables][1]
library in Symfony applications.

[1]: https://datatables.net

## Requirements 
- PHP 8.2 or higher
- StimulusBundle
- Composer

## Installation

Install the library via Composer:

```console
composer require pentiminax/ux-datatables
```
