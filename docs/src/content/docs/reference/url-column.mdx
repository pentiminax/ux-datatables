# UrlColumn

`UrlColumn` renders a cell as an `<a>` tag. Use it when the displayed value is a label or URL and the
link target can be read from the row, generated from a Symfony route, or computed with a callable.

## Raw URL Value

When the cell value already contains the URL, no extra configuration is needed:

```php
use Pentiminax\UX\DataTables\Column\UrlColumn;

UrlColumn::new('website', 'Website');
```

## Symfony Route per Row

Use `linkToRoute()` when the URL should be generated from a Symfony route. The callable receives the
same source row shape as action URL callables:

- with `AbstractDataTable` providers or `setData($objects)`, it receives the original object
- with array rows, it receives the source array

```php
use App\Entity\User;
use Pentiminax\UX\DataTables\Column\TextColumn;
use Pentiminax\UX\DataTables\Column\UrlColumn;

yield TextColumn::new('id', 'ID');
yield UrlColumn::new('email', 'Email')
    ->linkToRoute('admin_user_show', static fn (User $user): array => [
        'id' => $user->getId(),
    ]);
```

The displayed text stays the column value (`email` in this example). Only the `href` is generated.

## Custom URL per Row

Use `linkToUrl()` for external URLs or custom URL logic:

```php
use App\Entity\User;
use Pentiminax\UX\DataTables\Column\UrlColumn;

yield UrlColumn::new('profileLabel', 'Profile')
    ->linkToUrl(static fn (User $user): string => 'https://example.com/u/'.$user->getSlug())
    ->openInNewTab();
```

`linkToUrl()` also accepts a static string:

```php
UrlColumn::new('support', 'Support')
    ->setDisplayValue('Contact support')
    ->linkToUrl('/support');
```

Unsafe `javascript:` and `data:` URLs are rendered as escaped plain text instead of links.
